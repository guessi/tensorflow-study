# TensorFlow Study

## Prerequisite

- [pyenv](https://github.com/pyenv/pyenv)
- [pipenv](https://github.com/pypa/pipenv)

### Lesson 0 - Environment Setup

Switch to Python 3.8.6

    pyenv local 3.8.6

Make sure you are running correct version of Python

    python --version

Create virtualenv

    pipenv shell

Setup dependencies

    pipenv sync
